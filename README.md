# 易宿酒店预订平台 🏨

**第五期携程训练营 · 第 22 组大作业**  
本项目为小组协作完成，整体分为三大部分（用户端 / 服务端 / PC管理端）

## **移动端**
- **框架**
  - Taro（多端开发框架）
  - React

- **构建工具**
  - Webpack  
  > 使用 Taro 编译，将同一套代码构建为 H5 / 小程序等多端产物

- **UI 组件库**
  - Taro UI

- **状态管理**
  - Zustand

- **样式方案**
  - Less（CSS 预处理器）

---
